# Todo • Planner & Performance Dashboard

React + Vite + TypeScript + Tailwind. Tracks tasks, expenses, notes, calendar and performance charts.

## Scripts

```bash
npm install
npm run dev      # http://localhost:5173
npm run build    # production build
npm run preview  # preview build
```

## Features

- To‑do list with categories, due date/time and statuses (todo / in-progress / done)
- Week schedule and monthly calendar
- Expense tracking (daily / monthly / misc) with totals
- Performance charts (daily, weekly, monthly) using Recharts
- Notes with localStorage persistence

All data is stored locally in `localStorage` – no backend required.
